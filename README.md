# SimplCity-Landing-Page-Node
Landing page for SimplCity (localhosted... for now!)

1) Create a folder
2) git init
3) git pull https://github.com/dancrisan/SimplCity-Landing-Page-Node.git
4) npm install
5) node server.js
6) Open your fav browser and redirect to localhost:3000
